## วัตถุประสงค์

- **เพื่อพัฒนาระบบ Web Application ที่สามารถจำแนกสายพันธุ์แมวและนับจำนวนแมวในภาพได้ด้วยเทคโนโลยี AI**
- **เพื่อรวบรวมและให้บริการข้อมูลสำคัญเกี่ยวกับแมว** ได้แก่ ราคาประเมิน, นิสัย, อาหารที่เหมาะสม และความเหมาะสมในการเลี้ยงในไทย
- **เพื่อสร้างระบบแผนที่ (Cat Map) และคอลเล็กชัน** สำหรับบันทึกประวัติการพบเจอแมวของผู้ใช้งาน
- **เพื่อพัฒนาระบบจัดการข้อมูลสำหรับผู้ดูแลระบบ (Admin)** ในการจัดการคลาสข้อมูลแมวที่ไม่รู้จัก (Unknown Classes) เพื่อปรับปรุงโมเดลในอนาคต

## Dataset

ข้อมูลที่ใช้ในการฝึกโมเดลมาจาก [Roboflow Cat Dataset](https://universe.roboflow.com/project-gjxnt/cat-ccike/dataset/5)

| ชุดข้อมูล | สัดส่วน | จำนวนภาพ |
|----------|---------|----------|
| Train Set | 70% | 549 ภาพ |
| Valid Set | 20% | 156 ภาพ |
| Test Set | 11% | 83 ภาพ |

**รวมทั้งหมด: 788 ภาพ**

## คุณสมบัติหลัก

- 🔍 **จำแนกสายพันธุ์แมว** - ระบุสายพันธุ์แมวจากภาพที่อัปโหลด
- 📊 **นับจำนวนแมว** - นับจำนวนแมวที่ปรากฏในภาพ
- 📋 **ข้อมูลแมว** - แสดงข้อมูลราคา นิสัย อาหาร และความเหมาะสมในการเลี้ยง
- 🗺️ **Cat Map** - แผนที่บันทึกตำแหน่งที่พบแมว
- 📚 **คอลเล็กชัน** - เก็บประวัติการพบเจอแมวของผู้ใช้
- ⚙️ **ระบบ Admin** - จัดการข้อมูลแมวที่ไม่รู้จักเพื่อปรับปรุงโมเดล
